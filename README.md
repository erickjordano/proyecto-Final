Proyecto Final - Cotizador de Seguros TK-U
Este programa es un cotizador automático diseñado para la empresa de seguros TK-U, con el fin de agilizar el proceso de generación de cotizaciones que anteriormente se realizaba de forma manual.

Mejoras recomendadas
Validación de datos: Actualmente, si el usuario ingresa letras en lugar de números en el campo de "Edad" o "Salario", el programa podría dar un error. Sería ideal agregar una función que verifique el tipo de dato.
Interfaz Visual: Cambiar los alert y prompt por un formulario en HTML para que el usuario tenga una experiencia más cómoda y profesional.
Manejo de errores: Agregar mensajes más específicos cuando el usuario deja campos vacíos o ingresa datos inválidos.
